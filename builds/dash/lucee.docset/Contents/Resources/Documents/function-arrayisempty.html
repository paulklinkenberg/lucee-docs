ArrayIsEmpty

  • Home

Determines if the array is empty.

Returns: boolean

Usage

ArrayIsEmpty( array )

Arguments

Argument Description
array
(array, required)

Name of an array

Examples

myArray1 = array("Marcus","Sarah","Josefine");
myArray2 = arrayNew(1);

dump(var=arrayIsEmpty(myArray1),label="is myArray1 empty?");

dump(var=arrayIsEmpty(myArray2),label="is myArray2 empty?");
