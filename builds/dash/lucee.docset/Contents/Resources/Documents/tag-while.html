<cfwhile>

  • Home

  • <cfwhile>

Simplification of the tag cfloop-condition, analog to the cfscript "while" loop.

Usage

<cfwhile
    [condition="boolean"]
    [label="string"]
>

This tag is also supported within cfscript

Attributes

Attribute Description
condition
(boolean, optional)

condition o the expression
label
(string, optional)

used to a address this loop from a "break" or "continue" statement (instead of the nearest one).

Examples

There are currently no examples for this tag.
