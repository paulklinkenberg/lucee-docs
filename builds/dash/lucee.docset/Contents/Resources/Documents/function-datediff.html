DateDiff

DateDiff

Determines the number of given date parts between two different dates.

Returns: number

Usage

DateDiff( datepart, date1, date2 )

Arguments

Argument Description
datepart
(string, required)

The specific units in which to count:

  • yyyy: Years
  • q: Quarters
  • m: Months
  • y: Days of year (same as d)
  • d: Days
  • w: Weekdays (same as ww)
  • ww: Weeks
  • h: Hours
  • n: Minutes
  • s: Seconds
date1
(datetime, required)

date object; for example, now()
date2
(datetime, required)

date object; for example, now()
