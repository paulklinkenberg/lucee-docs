Saves or updates data of the entity and all related entities to the database. EntitySave(entity, [forceinsert])

Returns: void

Usage

EntitySave( entity [, forceInsert ] )

Arguments

Argument Description
entity
(object, required)

Name of the entity that must be saved in the database.
forceInsert
(boolean, optional)

If true, then Lucee always tries to insert the entity as a new record.

Examples

There are currently no examples for this function.
