ArrayMap

ArrayMap

  • Home
  • Reference
  • Lucee functions
  • ArrayMap

Calls the given closure with every element in the given array. the function returns a array that contains all values returned by the closure.

Returns: array

Usage

ArrayMap( array, closure [, parallel [, maxThreads ] ] )

Arguments

Argument Description
array
(array, required)

array to iterate
closure
(function, required)

filter can be a function/closure that implements the following constructor [function(any value[, number index, array array]):any].
parallel
(boolean, optional)

execute closures parallel
maxThreads
(number, optional)

maximum number of threads executed, ignored when argument "parallel" is set to false

Examples

There are currently no examples for this function.
