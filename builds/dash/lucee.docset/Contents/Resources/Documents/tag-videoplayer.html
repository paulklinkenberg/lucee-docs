<cfvideoplayer>

<cfvideoplayer>

  • Home
  • Reference
  • Lucee tags
  • <cfvideoplayer>

The tag CFVIDEOPLAYER allows you to display a video in flash (flv) format. It includes the complete functionality to display the player with the necessary html code. Read more on the multimedia functionality.

Usage

<cfvideoplayer
    [passthrough="struct"]
    [preview="string"]
    [id="string"]
    [video="string"]
    [autostart="boolean"]
    [group="boolean"]
    [playlist="string"]
    [playlistSize="number"]
    [playlistThumbnails="boolean"]
    [showPlay="string"]
    [showPause="string"]
    [showTimeline="string"]
    [width="string"]
    [height="string"]
    [foregroundcolor="string"]
    [foreground="string"]
    [fgcolor="string"]
    [bgcolor="string"]
    [backgroundcolor="string"]
    [background="string"]
    [screencolor="string"]
    [lightcolor="string"]
    [debug="boolean"]
    [allowfullscreen="boolean"]
    [linktarget="string"]
    [target="string"]
    [linkfromdisplay="boolean"]
    [overstretch="string"]
    [download="boolean"]
><!--- body --->[</cfvideoplayer>]

This tag may have a body.

This tag is also supported within cfscript

Attributes

Attribute Description
passthrough
(struct, optional)

attribute to pass natively vales to player
preview
(string, optional)

set this to the path of a preview image.
id
(string, optional)
video
(string, optional)

path to Movie File (flv/mp4)
autostart
(boolean, optional)

Set this to true in the player to automatically start playing when the page loads
group
(boolean, optional)

if multiple video are defined, you have to define if this videos should be break a part or played as a single video
playlist
(string, optional)

define if and where the playlist will be displayed
playlistSize
(number, optional)

size of the playlist, if playlist is right of display this is the width otherwise the height
playlistThumbnails
(boolean, optional)

using Thumbnails in playlist or not
showPlay
(string, optional)
showPause
(string, optional)
showTimeline
(string, optional)
width
(string, optional)
height
(string, optional)
foregroundcolor
(string, optional)

alias for fgcolor
foreground
(string, optional)

alias for fgcolor
fgcolor
(string, optional)

foreground color
bgcolor
(string, optional)

background color
backgroundcolor
(string, optional)

alias for bgcolor
background
(string, optional)

alias for bgcolor
screencolor
(string, optional)
lightcolor
(string, optional)
debug
(boolean, optional)
allowfullscreen
(boolean, optional)
linktarget
(string, optional)
target
(string, optional)
linkfromdisplay
(boolean, optional)
overstretch
(string, optional)

Sets how to stretch images/movies to make them fit the display. Set this to true to stretch them proportionally to fill the display, fit to stretch them disproportionally and none to keep original dimensions
download
(boolean, optional)

Set this to true to show a download button in the player controlbar
