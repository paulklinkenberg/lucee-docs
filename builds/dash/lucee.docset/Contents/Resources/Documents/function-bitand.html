BitAnd

BitAnd

  • Home
  • Reference

  • BitAnd

Performs a bitwise logical AND operation.

Returns: number

Usage

BitAnd( number1, number2 )

Arguments

Argument Description
number1
(number, required)

a integer
number2
(number, required)

a integer

Examples

The following code example will output an example numbers returned from the bitAnd() function.

<cfset inputValue1 = 5>
<cfset inputValue2 = 255>
<cfoutput>#bitAnd(inputValue1, inputValue2)#</cfoutput>
<br>
<cfset inputValue1 = 5>
<cfset inputValue2 = 0>
<cfoutput>#bitAnd(inputValue1, inputValue2)#</cfoutput>
