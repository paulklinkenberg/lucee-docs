Absolute-value function. The absolute value of a number is the number without its sign.

Returns: number

Usage

Abs( number )

Arguments

Argument Description
number
(number, required)

Number to find the absolute value of

Examples

absVal = abs( 1 ); 
echo( absVal ); // Outputs 1

absVal = abs( 1.1 );
echo( absVal ); // Outputs 1.1


absVal = abs( -1 );
echo( absVal ); Outputs 1


absVal = abs( -1.1 );
echo( absVal ); // Outputs 1.1
