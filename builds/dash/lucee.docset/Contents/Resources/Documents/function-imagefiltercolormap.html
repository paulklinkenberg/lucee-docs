ImageFilterColorMap

ImageFilterColorMap

  • Home

These are passed to the function ImageFilters (see ImageFilter documentation) which convert gray values to colors.

Returns: any

Usage

ImageFilterColorMap( type [, lineColor1 [, lineColor2 ] ] )

Arguments

Argument Description
type
(string, required)

type of the color map, possible values are: - spectrum: A colormap with the colors of the spectrum - linear: A colormap which interpolates linearly between two colors - grayscale: A grayscale colormap. Black is 0, white is 1
lineColor1
(string, optional)

used only for type "linear", the color corresponding to value 0 in the colormap.
lineColor2
(string, optional)

used only for type "linear", the color corresponding to value 1 in the colormap.

There are currently no examples for this function.
