FileSetAttribute

  • Home

For the given path, sets the file attributes.

Returns: void

Usage

FileSetAttribute( source, attribute )

Arguments

Argument Description
source
(any, required)

file path
attribute
(string, required)

One of the following: - readOnly - hidden - normal Set the attribute to normal to make a file not read-only and not hidden.
