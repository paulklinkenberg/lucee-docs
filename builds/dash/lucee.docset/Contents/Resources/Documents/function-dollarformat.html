Formats a string in U.S. format. (For other currencies, use LSCurrencyFormat)

Returns: string

Usage

DollarFormat( number )

Arguments

Argument Description
number
(string, required)

Number to format

Examples

<cfoutput>#dollarFormat(4.5)#</cfoutput>
