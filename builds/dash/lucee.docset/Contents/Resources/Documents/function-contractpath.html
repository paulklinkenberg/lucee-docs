ContractPath

  • Home

  • ContractPath

Does the opposite of the function expandPath(). Tries to match given absolute path to the mappings in the enviroment. When no mappings match, lucee creates a relative path from the current position.

Returns: string

Usage

ContractPath( path [, placeholder ] )

Arguments

Argument Description
path
(string, required)

a absolute path
placeholder
(boolean, optional)

If true (default false), Lucee also tries to create a placeholder for the begining of the Tag

There are currently no examples for this function.
