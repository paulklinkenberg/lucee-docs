ImageWriteBase64

ImageWriteBase64

  • Home

Writes Base64 images to the specified filename and destination.

Returns: string

Usage

ImageWriteBase64( name, destination, format [, inHTMLFormat ] )

Arguments

Argument Description
name
(any, required)

The image on which this operation is performed.
destination
(string, required)

The path for the destination file.
format
(string, required)

the image format
inHTMLFormat
(boolean, optional)

Specify whether Base64 output includes the headers used by the Base64 images in the HTML "img" tag ("data:image/{format};base64,...") : - true - false (default)

Examples

There are currently no examples for this function.
