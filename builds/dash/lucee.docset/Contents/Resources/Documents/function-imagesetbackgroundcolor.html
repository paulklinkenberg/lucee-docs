ImageSetBackgroundColor

ImageSetBackgroundColor

  • Home

  • ImageSetBackgroundColor

Sets the background color for the image. The background color is used for clearing a region. Setting the background color only affects the subsequent ImageClearRect calls

Returns: void

Usage

ImageSetBackgroundColor( name, color )

Arguments

Argument Description
name
(any, required)

The image on which this operation is performed.
color
(string, required)

Background color (The default value is "black"):

  • Hexadecimal value of RGB color. For example, specify the color white as ##FFFFFF or FFFFFF.
  • String value of color (for example, "black", "red", "green").
  • List of three numbers for (R,G,B) values. Each value must be in the range 0-255.

Examples

There are currently no examples for this function.
