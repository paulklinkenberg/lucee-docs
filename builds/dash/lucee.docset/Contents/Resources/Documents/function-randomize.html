Seeds the pseudo-random number generator with an integer number, ensuring repeatable number patterns.

Returns: number

Usage

Randomize( number [, algorithm ] )

Arguments

Argument Description
number
(number, required)

a number
algorithm
(string, optional)

The algorithm to use to generated the random number.
