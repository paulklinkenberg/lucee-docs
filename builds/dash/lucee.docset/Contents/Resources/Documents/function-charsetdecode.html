Converts as string to a binary representation.

Returns: binary

Usage

CharsetDecode( encoded_binary, encoding )

Arguments

Argument Description
encoded_binary
(string, required)

A string containing data to encode in binary format.
encoding
(string, required)

A string specifying encoding of the input data.
