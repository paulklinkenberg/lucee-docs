BinaryDecode

BinaryDecode

Converts a string to a binary object. Used to convert binary data that has been encoded into string format back into binary data.

Returns: binary

Usage

BinaryDecode( encoded_binary, binaryencoding )

Arguments

Argument Description
encoded_binary
(string, required)

A string containing encoded binary data.
binaryencoding
(string, required)

A string specifying the algorithm used to encode the original binary data into a string; must be one of the following: - hex: characters 0-9 and A-F represent the hexadecimal value of each byte - UU: data is encoded using the UNIX UUencode algorithm - base64: data is encoded using the Base64 algorithm

Examples

There are currently no examples for this function.
