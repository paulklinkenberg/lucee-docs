this function is experimentell, to not use in productive enviroment. this function returns the size in bytes of a given object.

Returns: any

Usage

SizeOf( obj [, complex ] )

Arguments

Argument Description
obj
(object, required)

object to get size for
complex
(boolean, optional)

if set to false, the size is returned, if set to true a struct containing the element count and the size is returned

There are currently no examples for this function.
