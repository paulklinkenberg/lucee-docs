Ceiling

Determines the closest integer that is greater than a specified number.

Returns: number

Usage

Ceiling( number )

Arguments

Argument Description
number
(number, required)

A real number

Examples

The following code example will output an example numbers returned from the ceiling() function.

<cfoutput>
	1.2: #ceiling(1.2)#<br>
	1.5: #ceiling(1.5)#<br>
	1.7: #ceiling(1.7)#<br>
</cfoutput>
