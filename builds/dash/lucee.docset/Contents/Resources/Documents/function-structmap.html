StructMap

StructMap

  • Home
  • Reference

  • StructMap

Calls the given closure with every element in the given struct. the function returns a struct that contains all values returned by the closure.

Returns: struct

Usage

StructMap( struct, closure [, parallel [, maxThreads ] ] )

Arguments

Argument Description
struct
(struct, required)

struct to iterate
closure
(function, required)

filter can be a function/closure that implements the following constructor [function(string key, any value[, struct struct]):any].
parallel
(boolean, optional)

execute closures parallel
maxThreads
(number, optional)

maximum number of threads executed, ignored when argument "parallel" is set to false

Examples

There are currently no examples for this function.
